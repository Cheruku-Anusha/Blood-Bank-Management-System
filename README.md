# Blood Bank Management System

A web-based/desktop application designed to efficiently manage blood donations, donor records, blood stock availability, and hospital requests. This system helps bridge the gap between blood donors and recipients.

## 🚀 Features
* **Donor Management:** Register donors, track donation history, and manage medical eligibility.
* **Blood Stock Inventory:** Real-time tracking of available blood units categorized by blood type (A+, A-, B+, B-, AB+, AB-, O+, O-).
* **Request & Distribution System:** Manage requests from hospitals or individuals and track approval status.
* **Admin Dashboard:** Centralized control panel for system administrators to generate reports and manage users.
